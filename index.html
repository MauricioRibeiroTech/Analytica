Project Overview

Comprehensive Dynamical Systems Analysis

This suite provides a complete workflow for studying coupled nonlinear systems, combining Julia's high-performance numerical computing with Python's rich ecosystem for data analysis and visualization.

The toolkit focuses on analyzing basin structures, boundary complexity, and topological entropy in systems of coupled oscillators with periodic forcing.
